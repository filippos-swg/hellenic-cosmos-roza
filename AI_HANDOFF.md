## Goal

Produce a visual identity exploration that makes "Ancient intelligence becoming active again" visible and credible.

A client looking at this should feel: this institution has become coherent. Not: they did a logo refresh.

## Output Format

Nine Phase 1 deliverables (ref: docs/CREATIVE_DIRECTION.md):

1. **Logo direction** — evolved four-square grid: refined mark, updated line quality, symbol logic
2. **Colour system** — primary palette rendered, optional accents shown
3. **Typography recommendation** — primary serif + secondary sans, with example settings
4. **Core icon set** — four quadrant icons (Ideas / Craft / Polis / Cosmos), square-framed, line-based, monochrome
5. **Programme icon set** — six pillar icons (Exhibitions / Tholos / Learning / Live / Lab / Digital)
6. **Website navigation example** — grid logic applied to nav structure
7. **Signage / ticket / poster example** — one printed application
8. **Brand-in-use board** — one composed visual showing the system working together
9. **Motion behaviour** — one written description or simple storyboard showing how the grid moves

Format: visual files + short written rationale for each direction decision.
