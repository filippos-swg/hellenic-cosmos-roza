## Decision Log

### 2026-05-25 — Working Platform Adopted

**Owner:** Filippos Arvanitakis / South North
**Status:** Active — working platform, not final approved copy

**Decision:**
Adopt "Civilization on the Move / Πολιτισμός εν Κινήσει" as the working strategic platform for Hellenic Cosmos.

**Why:**
Frames civilization as alive and moving, not preserved. Works in English and Greek. Avoids the "ancient meets future" cliché. Gives permission for technology, craft, migration, memory, language and future culture simultaneously.

**Caveat:**
Requires testing with native Greek copy sensitivity, brand identity and institutional tone before final confirmation.

### 2026-05-25 — Strategic Positioning: Own a Distinct Lane

**Owner:** Filippos Arvanitakis / South North
**Status:** Active

**Decision:**
Position Hellenic Cosmos as a future-facing cultural institution where Greek civilization is a living system. Not a venue with improved marketing.

Competitive lane: not Onassis (contemporary cultural intelligence), not SNFCC (civic infrastructure). Own: Greek civilization as a living cultural system — history, myth, science, democracy, theatre, philosophy, navigation, language, craft — not as archives but as active cultural forces.

**Why:**
The existing ingredients are strong. The failure is connection, not content. The work is not reinvention — it is making the ecosystem visible as a single coherent worldview.

### 2026-05-25 — Brand Move: Evolution, Not Erasure

**Owner:** Filippos Arvanitakis / South North
**Status:** Active

**Decision:**
Keep the existing four-square logo logic. Rebuild the brand system around it, not over it.

The four-square grid becomes the foundation for: navigation, signage, programme categories, exhibition identity, motion behaviour and digital UI. The mark evolves from static badge to living symbolic grid.

**Alternatives considered:**
Full rebrand / new mark.

**Why:**
The existing logo contains useful institutional DNA — modular grid, symbolic system, Greek references, recognisable public memory. The problem is dated execution, not the concept. The smarter move: unlock the system already inside it.

### 2026-05-25 — Ecosystem Architecture Locked

**Owner:** Filippos Arvanitakis / South North
**Status:** Active

**Decision:**
Organise Hellenic Cosmos under six named programme pillars:
1. Exhibitions — major shows and immersive experiences
2. Tholos — signature projection / VR / dome platform
3. Learning — schools, families, educational experiences
4. Live — talks, debates, performances, public programming
5. Lab — artists, engineers, researchers, prototypes
6. Digital — online experiences, archive, films, interactive content

**Why:**
Makes the institution understandable, programmable, sponsorable and communicable. Each pillar has a distinct function and audience path. Creates operational coherence and financial logic.

### 2026-05-25 — Programming Model: Ancient Questions, Future Formats

**Owner:** Filippos Arvanitakis / South North
**Status:** Active

**Decision:**
Use an annual theme model: one powerful cultural question generates programming across all six pillars in a connected cultural season.

Candidate first themes: Fate / Polis / Journey / Memory / Form / Human

**Why:**
Financial logic — one idea feeds many outputs. Brand logic — makes the institution feel like a coherent world, not an event calendar. Mirrors how strong cultural institutions operate.

### 2026-05-25 — Two Hero Pilot Concepts Selected

**Owner:** Filippos Arvanitakis / South North
**Status:** Active

**Decision:**
Develop two cultural experiment concepts as Phase 1 hero pilots:

1. **Ask the Oracle** — myth, AI, fate, probability and uncertainty. Connects ancient oracle tradition to contemporary AI and public participation. Interaction: visitor asks a question → system responds through voice, projection, symbolic fragments → visitor leaves with a reflection or digital fragment.

2. **The Living Loom / Weave the Cosmos** — loom as ancient computer, folk knowledge as future interface. Inspired by the Greek αργαλειός. Tactile, material, collective. Weaving as code, memory and cultural transmission.

These two concepts are deliberately complementary: The Oracle is intellectual and future-facing; The Living Loom is material, tactile and rooted in women's knowledge.

**Not yet decided:**
Which pilot to develop into a full Phase 2 proposal. Recommendation: develop both one level deeper before choosing.

### 2026-05-25 — Phase Structure Adopted

**Owner:** Filippos Arvanitakis / South North
**Status:** Active

**Decision:**
Three-phase structure:
- Phase 1 — Clarity: positioning + identity direction + digital prototype + two pilot concepts
- Phase 2 — Proof: one developed pilot + website prototype + audience and sponsor logic
- Phase 3 — System: full programming model + partnerships + content engine + sponsorship roadmap

**Why:**
Sequences ambition. Credible path without overcommitting in Phase 1.
