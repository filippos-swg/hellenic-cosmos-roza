## Goal

Produce a Phase 1 visual brand identity exploration for Hellenic Cosmos.

Make "Ancient intelligence becoming active again" visible and testable.
Strong enough to show a client. Not so finished it forecloses iteration.

---

## Context

The strategic direction is locked (see DECISIONS.md). The design system is defined: Grid + Fragment + Route + Signal built from the existing four-square logo (see docs/CREATIVE_DIRECTION.md). This is the first visual work in the project. It is direction, not final guidelines.

One unresolved decision affects this task: four quadrant naming (Ideas/Craft/Polis/Cosmos vs Logos/Techne/Polis/Cosmos). Work with Ideas/Craft/Polis/Cosmos as the default. Flag this clearly for client review before the icon system is finalised.

---

## Instructions

1. Read: PROJECT_BRIEF.md, DECISIONS.md, docs/CREATIVE_DIRECTION.md
2. Start with the logo evolution — the four-square grid is the foundation everything else builds from
3. Develop colour and typography in parallel once the grid logic is established
4. Build the icon system after the grid logic is confirmed
5. Compose the brand-in-use board last — it should prove the system, not invent it
6. Write a short rationale (2–4 sentences) for each direction decision

---

## Deliverables

1. Logo direction — evolved four-square mark (refined symbols, updated line quality, improved execution)
2. Colour system — primary palette rendered; optional accents shown separately
3. Typography recommendation — primary serif + secondary sans with example settings
4. Core icon set — Ideas / Craft / Polis / Cosmos (square-framed, line-based, monochrome)
5. Programme icon set — Exhibitions / Tholos / Learning / Live / Lab / Digital
6. Website navigation example — grid logic applied to nav structure
7. Signage / ticket / poster example — one printed application in system
8. Brand-in-use board — one composed visual showing the system working together
9. Motion behaviour — written description or simple storyboard for one grid movement

---

## Constraints

- Evolve the existing four-square logo — do not replace it
- No fake ancient Greek fonts, no generic tech fonts, no tourist-palette blue/white
- System must work across: institutional print, signage, digital UI, exhibition campaign, motion
- This is exploration and direction — not final brand guidelines
- Flag the four quadrant naming as unresolved before any icon system goes to client
